Getting Started with APIs

Learn how to integrate Wolfronix into your application

1

Navigate to API Page

Access your API management dashboard from the sidebar

2

Generate API Key

Click "Generate New Key" and copy it securely

Example key:

wfx_live_8h4k2j9d8f7g6h5j4k3l2m1n0
3

Make Your First API Call

Use your API key in the Authorization header

curl -X POST https://api.wolfronix.com/v1/scan \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"data": "sensitive information"}'
4

Monitor Usage

SDK Integration

Connect your application in minutes

1

Install SDK

Node.js / JavaScript

npm install @wolfronix/sdk

Python

pip install wolfronix
2

Initialize Connection

// JavaScript Example
const Wolfronix = require('@wolfronix/sdk');

const client = new Wolfronix({
  apiKey: process.env.WOLFRONIX_API_KEY,
  environment: 'production'
});

await client.scan({ data: 'sensitive information' });
3

Test Connection

const status = await client.health();
console.log(status); // { connected: true, latency: 45ms }
